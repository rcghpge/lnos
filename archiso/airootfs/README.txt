LnOS - Custom Arch Linux Install CD

This is the LnOS installation disc based on Arch Linux.

To install:
1. Boot from this disc
2. Run the LnOS installer when prompted
3. Follow the installation wizard

For more information, visit:
https://github.com/uta-lug-nuts/LnOS

System Requirements:
- x86_64 compatible system
- Minimum 1GB RAM
- 20GB+ disk space
